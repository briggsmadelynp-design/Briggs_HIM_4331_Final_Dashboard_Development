# visualization 1: trend of discharges over time (line plot)
# This shows whether hospital discharges increased or decreased over the 2013-2022 period
# explores the volume of patients
# I grouped data by year and added up the total number of discharges

# define the function so I can use it later
def plot_discharges_over_time(df):

    # aggregate the total discharges by year
    yearly = df.groupby("Year")["Tot_Dschrgs"].sum().reset_index()

    # create figure and axis object
    fig, ax = plt.subplots(figsize=(6,4)) #figsize (6,4) for consistency across dashboard

    # line plot
    ax.plot(yearly["Year"], yearly["Tot_Dschrgs"], marker='o', color='pink') # pink plots added for creativity and visual appeal

    # chart titles and varibles
    ax.set_title("Hospital Discharges by Year")
    ax.set_xlabel("Year")
    ax.set_ylabel("Total Discharges (In Millions)")

    # simple grid
    ax.grid (True)

    # return the function so that it can be used later on and display in the actual dashboard
    return pn.pane.Matplotlib(
        fig,
        tight=True,
        height=400,
        sizing_mode="stretch_width")

Observations:

The number of discharges shows a steady downward trend across the years, but around 2019-2020 there is a noticable drop in discharges. dishcarges fall from about 1.82 million in 2019 to 1.55 mil in 2020.

def plot_top10_avg_charges(df, column="Avg_Submtd_Cvrd_Chrg", top_n=10):

  # group diagnoses and calculate average charges
  avg_charges = df.groupby("DRG_Desc")["Avg_Submtd_Cvrd_Chrg"].mean().sort_values(ascending=False)

  top10 = avg_charges.head(10)

  # chart titles and varibles
  ax.set_xlabel("Average Submitted Covered Charges (In Millions $)")
  ax.set_ylabel("Diagnosis")
  ax.set_title("Average Charges for Top 10 Diagnoses")

  # simple grid
  ax.grid(True)

Filter design and reasoning¶

Design:

I will be applying 6 filters to interactivley update the data visualizations. My filters will include interactive widgets that can ajust year, diagnoses, hospital geographic area, Average total payment, average Medicare payment, along with discharge information. These filiters should be able to give more insight in describing my 4 visualizations.

Reasoning:

  • year dropdown - I chose the drop down selection too make it easy to select and view one year at a time.

  • diagnosis search - I decided on the text box widget to make it convenient to search for a specific diagnosis

  • geographic search - I decided on the text box widget to make it convenient to search for a specific state.

  • avg total payment slider - I chose the range slider for payments since it is filtering large numerical data.

  • avg medicare payment slider - I decided to do a range slider again because of the large numerical data being filtered

  • total discharges slider - I chose the range slider becuase it will help filter hospitals by the range of total discharges.

Limitations:

When first brainstorming how I should assign my filiters, I was planning on linking each filter to plots with coordinating variables. However, I soon realized that would not preoduce as meaning ful results because the data would not show enough varation when the filter was applied. to make the dashboard more interesting I went with a a plan to apply filters to plots with differeing variables to the plots, which showed more interesting results.

Another limitation that i discovered was while testing the interactivity of the histogram. At first, I had applied a geographic search box to filter by state but, it limited the filter to only one state at a time which hardly produced any variance in the chart. So, i decided to go with a multiselect box that way users could choose more than one state. this helped create more meaningful updates in the histogram when the filter was applied.

# now I will connect the widgets and filters to the plots

# so, I applied the filters with differeing variables to make sure the filters would produce more insightful results

# dynamic line plot with diagnosis multi select
@pn.depends(diagnosis_search)
def updated_discharges_plot(search_text):
    if not search_text.strip():  # if the search box is empty, show all diagnoses
        filtered_df = df
    else:
        filtered_df = df[df['DRG_Desc'].str.contains(search_text, case=False, na=False)] # filter by the diagnosis

    # return the plot
    return plot_discharges_over_time(filtered_df)

# dynamic bar plot with year dropdown selection
@pn.depends(year_dropdown)
def updated_bar_plot(selected_year):
    if selected_year == "ALL":
        filtered_df = df  # if nothing is selected, show all years
    else:
        filtered_df = df[df['Year'] == selected_year]  # filter by the year selected

    # return the plot
    return plot_top10_avg_charges(filtered_df)

# dynamic histogram with geographic location multi - select
@pn.depends(state_select)
def updated_histogram(selected_states):
    if not selected_states:
        filtered_df = df  # if nothing is selected, show all data
    else:
        filtered_df = df[df['Rndrng_Prvdr_Geo_Desc'].isin(selected_states)] # filter by the state selected

    # return the plot
    return plot_total_payment_histogram(filtered_df)

# dynamic scatter plot with total payment slider, total discharge slider, and total medicare payment slider
# I wanted to conbine these 3 filters to affect the scatter plot all at once
@pn.depends(total_payment_slider, discharges_slider, medicare_payment_slider)
def updated_scatter_combined(total_payment, discharges, medicare_payment):
    filtered_df = df[
        (df['Avg_Tot_Pymt_Amt'] >= total_payment[0]) & (df['Avg_Tot_Pymt_Amt'] <= total_payment[1]) &
        (df['Tot_Dschrgs'] >= discharges[0]) & (df['Tot_Dschrgs'] <= discharges[1]) &
        (df['Avg_Mdcr_Pymt_Amt'] >= medicare_payment[0]) & (df['Avg_Mdcr_Pymt_Amt'] <= medicare_payment[1])]

# return the plot
    return scatter_plot(filtered_df)
